# Лабораторная работа № [7]
## Обработка одномерных массивов в C++

### Цель работы
Изучение алгоритмов обработки массивов на языке C++, освоение принципов работы с указателями и разработка программы реверса массива.

### Задание
Разработать программу, выполняющую обращение (реверс) одномерного массива целых чисел.

### Реализация
Программа содержит:
1. Инициализацию тестового массива
2. Функцию реверса массива
3. Вывод результатов на экран

### Ключевые элементы кода
```cpp
// Функция реверса массива
void reverseArray(int arr[], int size) {
    for(int i = 0; i < size/2; i++) {
        swap(arr[i], arr[size-i-1]);
    }
}
